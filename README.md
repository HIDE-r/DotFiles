# Dotfile Management

This is my dotfiles project which I use [GitHub - anishathalye/dotbot: A tool that bootstraps your dotfiles ⚡️](https://github.com/anishathalye/dotbot) as manager.

To set up this project, just need one step:

```bash
make dotbot
```
